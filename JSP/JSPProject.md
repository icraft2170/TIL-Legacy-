# JSP

## JSP로 웹 프로그래밍 하기
- JSP를 이용하여 Servlet프로그램을 만든다. `Jasper`를 통해 JSP를 이용해 Servlet을 생성한다
- Jasper가 작성할 서블릿 코드에 코드블럭을 적절하게 끼어달라고 지시하는 방식


### 코드블럭
```jsp
<% %>
<!-- 스크립트릿 -->
<%= %>
<!-- 표현식 -->
<%! %>
<!-- 선언부 -->
<%@ %>
<!-- 디렉티브 -->
```
#### 필수
```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8" %>
```
